## Stack

- Vite 5 · React 19 · React Router
- Tailwind v4 · oklch dark theme
- @xyflow/react (grafo interactivo)
- recharts (gráficos)
- TypeScript · Zod

## Desarrollo

```bash
npm install
npm run dev
```

## Deploy

Push a `main` → GitHub Actions → GitHub Pages (automático).
